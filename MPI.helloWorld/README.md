# Purpose
MPI hello world project

# Installation
Prerequisites:

Install [MPICH](https://www.mpich.org/downloads/)
	>>> tar -xzf mpich-3-3.2.tar.gz
	>>> cd mpich-3-3.2
	>>> ./configure
	>>> ./configure --disable-fortran
	>>> make; sudo make install
	>>> mpiexe --version

# Execution
Execute on CLion for instance.
